Progressive Web App

LeapMultix fonctionne hors ligne en trois gestes

La PWA offre le meilleur des deux mondes : légèreté d’un site web et confort d’une application native. Aucun store à valider, aucune mise à jour à surveiller : les nouvelles versions sont poussées automatiquement via le Service Worker.

Installation rapide

Android

  1. Ouvrez LeapMultix dans Chrome ou Edge.
  2. Appuyez sur « Installer l’application » (icône flèche + écran).
  3. Validez : l’icône apparaît sur l’écran d’accueil.

iOS / iPadOS

  1. Ouvrez le site dans Safari.
  2. Menu « Partager » → « Ajouter à l’écran d’accueil ».
  3. Renommez si besoin et confirmez.

Windows & macOS

  1. Depuis Chrome/Edge, cliquez sur l’icône « Installer LeapMultix » dans la barre.
  2. Autorisez la création du raccourci.
  3. L’application se lance dans une fenêtre dédiée.

Chromebook

  1. Ouvrez LeapMultix.
  2. Dans l’omnibox, choisissez « Installer ».
  3. Déployez l’application via Google Admin si nécessaire.

Chaque installation embarque le cache nécessaire (jeux, sons, traductions). Après la première connexion, l’enfant peut jouer dans l’avion ou dans une salle sans wifi.

Avantages clés

  • Temps de chargement réduit (critical assets préchargés, lazy loading des modes).
  • Service Worker gère la mise à jour silencieuse : l’utilisateur est notifié lors d’une nouvelle version.
  • Paramètres persistants : thème, langue, volume et comptes enfants sont conservés même en mode avion.

Résolution des problèmes courants

  • L’option installer n’apparaît pas : assurez-vous d’utiliser un navigateur compatible PWA et que le site est chargé via HTTPS.
  • L’appli reste blanche hors ligne : ouvrez LeapMultix en ligne une fois par semaine pour rafraîchir le cache.
  • Besoin de libérer de l’espace : supprimez l’application depuis les réglages système, puis réinstallez-la. Les données locales peuvent être exportées via le menu parent.

En pratique

Une fois LeapMultix installée, l’icône se comporte comme n’importe quelle application native. Pensez simplement à lancer l’app en ligne une fois par semaine pour récupérer les dernières mises à jour.

Si vous administrez plusieurs appareils (classe, médiathèque), gardez à portée les étapes d’installation ci-dessus pour les reproduire rapidement. Tous les profils et paramètres restent stockés sur l’appareil des enfants.
