# LeapMultix Dev Audit

Pack conformité/audit : accessibilité, i18n, sécurité et vérifications PWA.

## Contenu

- **Commandes** : `/audit-config`
- **Agents** : `accessibility-auditor`, `i18n-coordinator`, `web-research-specialist`, `pwa-expert`
- **Skills** : `accessibility`, `auditing-security`, `dependency-management`, `i18n-sync`, `pwa-service-worker`

## Synchronisation

```bash
npm run plugin:sync -- --profile=audit
```
